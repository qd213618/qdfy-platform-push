# 青蝶飞云平台推送

**ID**: com.qdfy.platform-push
**版本**: 1.0.0
**作者**: qdfy
**许可**: MIT

## 功能描述

接收青蝶飞云协作平台推送的通知和问卷消息，在独立会话中实时展示。

Skill 每隔 30 秒轮询一次平台接口，有消息时自动在专属会话中弹出，无需服务器主动推送。

## 配置项

| 配置项 | 必填 | 说明 |
|--------|------|------|
| 服务器地址 | ✅ | 青蝶飞云协作平台的地址，如 `https://your-server.com:8765` |
| Push Token | ✅ | 在平台 `/platform → OpenClaw 推送` 页面生成，格式 `pk_xxxxxxxx` |
| 轮询间隔（秒） | | 默认 30 秒，范围 10～300 |

## 使用步骤

1. 用浏览器打开青蝶飞云协作平台，登录账号
2. 进入 `/platform` → 点击顶部「📡 OpenClaw 推送」标签页
3. 点击「生成 / 重新生成 Token」，复制 Token
4. 在 QClaw Skill 设置中填入服务器地址和 Push Token
5. 回到平台点击「📨 投入测试消息」验证连通（约 30 秒内收到）

## 权限说明

- **network**：用于轮询平台接口拉取消息
